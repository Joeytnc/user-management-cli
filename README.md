# user-management-cli

A simple command-line user management system built with Python.

## Features

- Add new user with name validation
- Age input validation (integer only)
- Delete user by ID
- List all users
- Defensive input handling

## Tech Stack

- Python 3
- CLI interaction
- List & dictionary data structures
- Exception handling
- Control flow design
